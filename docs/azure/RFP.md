# 「Transcript Minute」アプリケーション新規開発案件のご依頼

## 1. 概要

この度、AIを活用した高機能な文字起こし・議事録生成Webアプリケーション「Transcript Minute」の新規開発をご依頼します。
既存システムのコンセプトをベースに、プラットフォームをMicrosoft Azureへ刷新し、Azure OpenAI Serviceを活用したより強力な機能を実装することを目的としています。

つきましては、Azure上でのインフラ設計・構築から、アプリケーション開発、AI関連のチューニングまでを一括してお任せできる開発パートナー様を募集しております。

## 2. プロジェクトの目的と背景

音声や動画ファイルを手作業で文字起こしし、議事録を作成する業務は多くの時間を要します。本プロジェクトでは、このプロセスをAIによって自動化し、ユーザーの生産性を飛躍的に向上させるSaaSアプリケーションを開発することを目的とします。

## 3. 主な機能要件

開発をお願いしたいアプリケーションの主要な機能は以下の通りです。詳細は添付の要件定義書をご参照ください。

- **ファイルアップロード機能**: 多様な音声・動画ファイル形式に対応。
- **AIによる自動処理**:
    - **多言語文字起こし**: Azure OpenAI Service (Whisper) を利用。
    - **話者識別**: 文字起こし結果から話者を自動で分離・識別。
    - **コンテンツ生成**: 文字起こし結果を元に、箇条書きの要約、構造化された議事録、タスクリストを自動生成。
- **編集・管理機能**:
    - **話者名の編集**: ユーザーが手動で話者名を修正できる機能。
    - **リアルタイム進捗表示**: ファイルアップロードから処理完了まで、ステータスをリアルタイムでユーザーに通知。
    - **結果のダウンロード**: 生成されたコンテンツを一括でダウンロードする機能。
- **マルチテナント・セキュリティ**:
    - 組織単位でのユーザー管理とデータ分離。
    - ユーザーのロール（管理者、一般）に基づいたアクセス制御。

## 4. 開発範囲

本案件でご担当いただきたい作業範囲は以下の通りです。
なお、参考資料としてAWS上に構築された既存アプリケーションのソースコードをご提供可能です。設計の参考に、また転用可能な部分は積極的にご活用ください。

1.  **インフラ設計・構築**
    - アプリケーション稼働に必要となる全てのAzureインフラの設計と構築をお願いします。
    - 作業には、弊社が付与する開発用のAzureアカウントをご利用いただきます。
    - 想定サービス: Azure App Service, Azure Functions, Azure Blob Storage, Azure SQL Database/Cosmos DB, Azure AD B2C, Azure Key Vault, Azure OpenAI Service 等。

2.  **アプリケーション開発**
    - フロントエンド、バックエンド、データベースの設計と実装。
    - CI/CDパイプラインの構築とデプロイの自動化。

3.  **AI機能の組み込みと最適化**
    - Azure OpenAI Serviceをバックエンドに統合し、主要機能を実現してください。
    - **LLMモデルの選定**: 要件に最適なGPTモデルを選定・検証してください。
    - **プロンプトエンジニアリング**: 文字起こし後のコンテンツ生成（要約、議事録、タスクリスト）において、より高品質な出力が得られるよう、プロンプトの設計と改善を行ってください。工夫と改善提案を歓迎します。

## 5. 必須スキル・ご経験

ご応募いただくにあたり、以下のご経験を必須とさせていただきます。

- **Azureの広範なインフラ構築および開発経験**（IaaS, PaaS, Serverless問わず）
